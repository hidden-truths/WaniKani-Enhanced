Design notes

  1. 1
    Paste raw lyrics. No formatting, no furigana, no hand-markup — just the text. Mirrors how custom cards / Self-Talk phrases are authored, but the analysis (next) does the heavy lifting.
  2. 2
    Link the video. One YouTube URL is the audio source for everything downstream (synced playback, dictation, by-ear shadowing). Embedding sidesteps re-hosting copyrighted audio.
  3. 3
    Analyze = full-auto. One pass generates furigana, a per-line English translation, grammar tags, and a JLPT/difficulty profile (your “full auto” choice). This is the one genuinely new capability — there’s no runtime furigana/translate today.
  4. ·
    Privacy posture. BYO lyrics become private store rows (like custom-card examples); only the curated starter set is public. No copyrighted lyrics are redistributed.
